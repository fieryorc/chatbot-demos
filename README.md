# Chatbot Demos

This repository contains examples and demonstrations of various chatbots and AI agents.

## Contents

- `debate_agent/`: A debate orchestration using the `google.adk.agents` framework. It includes a `SequentialAgent` demonstrating a debate flow between a Researcher (Agent A), a Critic (Agent B), and a Judge (Agent C).

## Setup

Follow the steps in [google adk documentation](https://google.github.io/adk-docs/get-started/quickstart/) to
set up the python venv environment.

1. Make sure you have the required dependencies installed (e.g. `google.adk.agents`).
2. Configure your environment variables in `.env`.
3. Run the scripts or integrate the agents as needed.
